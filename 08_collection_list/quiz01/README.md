## 문제) List 기본 사용

### 1. List 만들기
* 리스트에 주어진 값들을 넣고 출력하세요.
```
3, 8, 9, 4, 2, 1, 7, 5
```

> 출력 예시
```
[3, 8, 9, 4, 2, 1, 7, 5]
```

### 2. 값 수정하기
* List의 네번째 값을 6으로 바꾸고 전체를 출력하세요.

> 출력 예시
```
[3, 8, 9, 6, 2, 1, 7, 5]
```

### 3. 값 삭제하기
* List에서 8과 7을 삭제하고 전체를 출력하세요.

> 출력 예시
```
[3, 9, 6, 2, 1, 5]
```
